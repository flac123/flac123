flac123 has moved to github

flac-tools consists of flac123, a command-line program for playing FLAC audio files.

Download

Latest stable release: flac123-0.0.12.tar.gz                        Older releases are here.

Requires: libao, FLAC, libpopt
Fedora users will need to install the libao-devel and flac-devel RPMs.
Debian/Ubuntu users need to install libao-dev, libflac-dev and libpopt-dev packages.

Tested on Ubuntu 12.04.
Older releases (0.0.11) on FreeBSD 6.2, Mac OS 10.6.

A PPA for Ubuntu can be found here.
